Desarrollador
Full Stack

Un desarrollador full stack es un profesional que trabaja en todos los aspectos del desarrollo de software, abarcando tanto el front-end (la parte visible con la que interactúa el usuario, usando lenguajes como HTML, CSS y JavaScript) como el back-end (la parte del servidor, que incluye bases de datos, lógica de aplicación y APIs). Esta versatilidad les permite crear aplicaciones web completas, desde la interfaz de usuario hasta la infraestructura del servidor

Acerca de mi

Soy Manuel Maza, Administrador de empresas de profesión, con experiencia en el rubro financiero, y comercial. Me gusta aprender del campo de la tecnología y mi gran desafio es ser experto Desarrollador Full Stack.

HTML Y CSS

Con HTML aprendí a definir el contenido y estructurar una página web, utilizando etiquetas, encabezados, párrafos, listas, y enlaces. Con CSS aprendí a controlar la presentación y el diseño de los elementos HTML, como color, fuente, tamaño, espaciado y otros.

FLEXBOX

Con Flexbox aprendí a alinear contenido, construir leyouts que contengan elementos distribuidos en filas y columnas, crear una galeria de imágenes, crear un menú y una seccion de características. Permite crear diseños de página flexibles y receptivos.

BOOTSTRAP

Con Bootstrap estoy aprendiendo a desarrollar sitios web, utilizando herramientas que permiten que el desarrollo sea rápido y reponsive, compatible con diferentes dispositivos y navegadores.
